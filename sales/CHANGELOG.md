# Changelog

All notable changes to this project are documented in this file.

## [0.1.3] - 2026-02-26

### Changed
- Enabled all bundled MCP connectors by default in mcp/config.json.
- Kept auto-registration behavior via plugin config hook so connectors appear automatically in OpenCode.

## [0.1.0] - 2026-02-23

### Added
- Initial standalone OpenCode plugin scaffolding in TypeScript.
- /sales command router and converted sales command workflows.
- Converted sales skill catalog for OpenCode package layout.
- Optional MCP connector catalog covering all source sales plugin connectors.
- Windows-native runtime state path and initialization behavior.
- Windows installation and configuration documentation.
