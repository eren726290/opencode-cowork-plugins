# Quick Reference

## Recommended Prompt Pattern

```text
/sales create-asset <Company>
Audience: <Executive|Technical|Mixed>
Goal: <What outcome you need>
Format: <Landing page|Deck|One-pager|Workflow demo>
Must include: <Top 2-3 requirements>
```

## Format Selection Guide

- Landing page: broad narrative + interactive sections
- Deck: linear meeting flow
- One-pager: concise leave-behind
- Workflow demo: technical sequence and integrations

## Final QA Before Sharing

- Names and titles are correct
- CTA is explicit
- Claims are evidence-backed
- Visual contrast/readability is good
